ABOUT US:           

    VEDA comes from "VEDere Artificială" (meaning ARTIFICIAL VISION in Romanian).

    In the last more than 30 years we focused our R&D work mainly on the following topics of the COMPUTER VISION field:

    • Image Processing, Image Analysis, Pattern Recognition
    • ANPR / ALPR - Automatic Vehicle License (Number) Plate Recognition
    • OCR / ICR - Optical / Intelligent Character Recognition
    • Automatic Optical / Video Surveillance

    Among other stuff, we also designed, developed and implemented some original proprietary models, methods, technologies and algorithms, some based on Artificial Neural Networks - ANN (i.e. neural classifier), or, later, some "not neural but somehow alike" ones (i.e. classifier, associative memory model). Some of them are incorporated in the software products enumerated below.

VEDA PRODUCTS:             

ANPR / ALPR - Automatic Number / License Plate Recognition

  • VEDA LPR   (Last update: Mar. 2020)   - Our Windows / 32-bit (running also on 64-bit) Automatic Vehicle Number / License Plate Recognition (ANPR / ALPR) software solution (demo application), based on the VEDA OCR/NeurOCR engine.
  • VEDA LPR - SDK / API   (Last update: Mar. 2020)   - Our LPR engine functions library (ALPR / ANPR SDK) - Windows / 32-bit DLLs (running also on 64-bit) -, and application programming interface (ALPR / ANPR API), for software developers / integrators of ANPR / ALPR applications.
  • VEDA CAR   (Just history!)   - An old (1993-1996) version of a general optical text reading software kernel, initially intended for real time vehicle registration plate recognition (DOS / 16-bit prototype, and Windows / 32-bit demo releases).

OCR / ICR - Optical / Intelligent Character Recognition

  • VEDA OCR / NeurOCR 3.0   (Last release: Nov. 2007 / Lsst update: May 2017)   - Our trainable, not neural but alike, OCR package (Windows / 32-bit, running also on 64-bit).
  • VEDA NeurOCR 2.0-d   (Just history!)   - An old (1996-2001) version of our Neural Networks based NeurOCR package (Windows / 16-bit, running also on 32-bit). Several even much older (1992-1995) releases of version 1.x for DOS (16-bit) existed too.

OTHER STUFF / PROJECTS

  • VEDA SimpleOpt   - A free, fully usable, general optimizer, Python implementation of an original method for solving various classes of optimization problems.
  • VEDA LPR for railway wagons UIC - SDK / API   - A customized VEDA LPR engine software functions library (Windows / 32-bit DLLs) for automatically "reading" inscriptions on railway wagons.
  • VEDA InvOCR   - An OCR application for variable/flexible layout, multi-page invoices "reading" (Windows / 32-bit).
  • VEDA FormOCR   - A prototype OCR application for fixed layout forms "reading" (Windows / 32-bit).
  • VEDA OCR / NeurOCR - SDK / API   - Our OCR / NeurOCR engine functions library (Windows / 32-bit DLLs).
  • VEDA SCENE WATCHER   (Just history!)   - An old (1997-2002) simple "proof of concept" prototype (Windows / 16-bit), simulation for a real time optical scene surveillance system.
